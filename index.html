Introduction

Une API (Application Programming Interface) est une interface par laquelle un logiciel offre des services à un autre logiciel. On peut accéder à ces services de deux manières : en installant une bibliothèque comme jQuery qui facilite les manipulations DOM en se connectant à un Web service comme OpenWeatherMap qui délivre plus d’un milliard de prévisions météo par jour Les APIs sont en principe accompagnées d’une documentation qui spécifie comment se servir des différentes fonctionnalités.

Web service et API Key

Pour accéder à un Web service on a souvent besoin d’une clé d’identification, appelée API Key. Cette identification permet au fournisseur de surveiller les accès et de révoquer la clé en cas d’abus. Pour les services payants, la clé sert également à facturer les accès. Certaines API comme flickr offrent un accès public sans identification. Pour obtenir une clé il suffit généralement de s’inscrire chez le fournisseur.
